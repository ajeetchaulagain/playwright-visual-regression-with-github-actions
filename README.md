# Visual Regression Testing Demo with Playwright & GitHub Actions

A demo project showcasing visual regression testing using [Playwright](https://playwright.dev/) in a [Next.js](https://nextjs.org/) application with automated CI/CD integration via GitHub Actions.

## Prerequisites

> [!IMPORTANT]
> This application requires **Node.js v18.17+.**

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/ajeetchaulagain/playwright-visual-regression-with-github-actions.git
cd playwright-visual-regression-with-github-actions
```

### 2. Install dependencies

```bash
npm install
```

### 3. Install Playwright browsers

```bash
npx playwright install
```

### 4. Run the development server

```bash
npm run dev
```

Your app should be running at [http://localhost:3000](http://localhost:3000)
